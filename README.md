## 🚀 About This Project

This repository is dedicated to honing logical thinking and mastering **Vanilla JavaScript (ES6+)** syntax. Everything is executed directly within the **Node.js** environment, stripping away complex UI frameworks to focus purely on the "heart" of JavaScript – the Core logic.

- 🧠 **Focus:** Algorithm logic & ES6+ Syntax.
- ⚡ **Environment:** Server-side execution with Node.js.
- 🛠️ **No Frameworks:** Pure JavaScript only.

## 🛠️ Prerequisites

To run this project locally, ensure you have the following versions installed:

- **Node.js:** `v25.9.0` (or latest)
- **NPM:** `v9.5.1` (or latest)
